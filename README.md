# Eldoria — Minecraft Server Website

A clean, [Modrinth](https://modrinth.com)-inspired homepage for the **Eldoria** server with live status, Dynmap, news, staff, and **LevelZ** player stats.

## Quick start

```bash
npm install
npm run dev
```

This starts both the **API** (port 3001) and **Vite** (port 5173). Open [http://localhost:5173](http://localhost:5173).

## Features

- **Live Players** — real-time count via mcstatus.io
- **Player Base** — total unique players who have joined (from synced data)
- **Player Stats** — LevelZ levels, XP, and skills for every player
- **Server Info** — version, gamemode, features
- **Live Dynmap** embed
- **News & Updates** — editable in config
- **Staff roster** — editable in config

## Syncing real LevelZ data from your server

LevelZ is a **Fabric mod** — there is no file access on Shockbyte, so use the **Eldoria Bridge** mod instead.

### Option A: Eldoria Bridge mod (recommended for Shockbyte)

1. Build or download `eldoria-bridge` from the `eldoria-bridge/` folder
2. Upload the JAR to your server's `mods/` folder (requires Fabric + LevelZ)
3. Open an extra port in Shockbyte (e.g. `26871`) and set it in `config/eldoria-bridge.json`
4. Point your website API at the mod:

```bash
BRIDGE_URL=http://YOUR_SERVER_IP:26871
BRIDGE_API_KEY=your-key-from-eldoria-bridge.json
npm run api
```

See **[eldoria-bridge/README.md](eldoria-bridge/README.md)** for full setup.

### Option B: Local file sync (VPS / dedicated only)

If you have filesystem access to the world folder:

```bash
WORLD_PATH=/path/to/your/world npm run sync
```

## Configuration

| File | Purpose |
|------|---------|
| `public/config.js` | Server IP, Dynmap, Modrinth, staff, news |
| `server/data/players.json` | Player + LevelZ stats (auto-generated by sync) |
| `.env` (optional) | `MC_HOST`, `MC_PORT`, `WORLD_PATH`, `API_PORT` |

## API endpoints

| Endpoint | Description |
|----------|-------------|
| `GET /api/server` | Live players, player base count, online status |
| `GET /api/players` | Full player list with LevelZ skills |
| `GET /api/health` | Health check |

## Production deploy

1. `npm run build` → deploy `dist/`
2. Run `npm run api` on your server (or proxy `/api` to it)
3. Set up a reverse proxy for `/dynmap` if using HTTPS
4. Schedule `npm run sync` on the game server

## LevelZ skills tracked

Mining, Strength, Defense, Agility, Stamina, Luck, Archery, Smithing, Farming, Health, Trade, Alchemy — plus overall **Level**, **XP**, and **skill points**.
